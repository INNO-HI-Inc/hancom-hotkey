버전 관리는 Semantic Versioning을 따릅니다.

v0.4.0 LATEST2026-04-16

Added

  • Sparkle 자동 업데이트: 24시간마다 새 버전 확인. 메뉴바·설정·정보 탭에서 수동 확인 가능.
  • 한셀·한쇼 지원: 한글(HWP)뿐 아니라 한셀·한쇼도 자동 감지 매핑.
  • 전역 토글 단축키: 어디서든 ⌃⌥⌘P로 매핑 일시정지·재개.
  • 슬립·웨이크 복구: 맥 슬립에서 깨어날 때 EventTap 자동 재등록.
  • 로그 내보내기: 메뉴바·설정 → 문제 해결에서 최근 1시간 로그를 .txt로 저장.
  • 메뉴바 드롭다운 개편: 상태 헤더, "HWP 스타일 순서 바꾸기" 바로가기, 업데이트·로그 메뉴 추가.
  • 랜딩 페이지: 제품 소개 + 다운로드 CTA 공식 사이트 오픈.

v0.3.0 2026-04-16

Changed

  • 핵심 단순화: 커스텀 매핑 편집 UI 제거 (⌘0~4 → ⌃2~6 고정 매핑 유지). HWP는 스타일 이름이 아니라 스타일 목록 순서로 단축키를 붙이므로, 앱에서 이름·타깃키를 편집하는 건 실사용에서 혼란만 야기.
  • 설정 → 단축키 탭을 읽기 전용 매핑 표 + HWP 스타일 순서 변경 5단계 가이드로 재구성.

Removed

  • KeyCaptureView, AddMappingSheet, HwpShortcutGuideView 등 커스텀 매핑 관련 UI 제거.

v0.2.3 2026-04-16

Fixed

  • 매핑 편집·추가가 실제로 반영되지 않던 치명적 버그. @Published sink 관찰자가 willSet 타이밍에 실행되어, UI는 새 값이지만 디스크 저장은 이전 값으로 되던 문제 수정 (didSet 기반 재작성).

v0.2.2 2026-04-16

Changed

  • "매핑 추가"를 인라인 삽입 대신 모달 시트 + 확인 버튼 방식으로 개선. 중복 매핑 검증 추가.

v0.2.1 2026-04-16

Fixed

  • 설정 → 단축키 탭에서 매핑 편집 시 Auto Layout 무한 제약 루프로 앱이 강제 종료되던 문제 수정.

v0.2.0 2026-04-16

Added

  • 커스텀 매핑 편집: 매핑 이름 인라인 편집, 입력/HWP 단축키 키 캡처, 매핑 추가/삭제. (v0.3.0에서 제거됨)

v0.1.0 2026-04-16

Added

  • HWP 활성 시 ⌘0~4 → ⌃2~6 자동 리매핑.
  • 메뉴바 상주 유틸리티 (NSStatusItem, LSUIElement).
  • 접근성 권한 온보딩 화면, 설정 윈도우 3탭(일반/단축키/정보).
  • 로그인 시 자동 실행, Secure Input 자동 비활성, EventTap 자동 재활성.
  • macOS 13+ Universal binary, Developer ID 서명 + 공증, DMG 배포.
