한컴단축키

Mac에서 한컴 오피스 스타일 단축키를 ⌘0~4로 편하게.

⬇︎ DMG 다운로드 릴리즈 기록

macOS 13 Ventura 이상 · Universal (Apple Silicon + Intel) · 서명 + 공증 완료

왜 필요한가

한컴 오피스 Mac판의 스타일 단축키는 Ctrl + 숫자 조합이라 맥 사용자에겐 어색합니다. 한컴단축키는 한글(HWP), 한셀, 한쇼가 활성일 때만 ⌘ + 0~4를 자동으로 Ctrl + 2~6으로 변환합니다. 다른 앱에서는 원래 단축키 그대로 동작합니다.

특징

▲ 네이티브 Swift

Hammerspoon 없이도 메뉴바에 가볍게 상주. CPU/메모리 부담 최소.

▲ 한컴 3종 지원

한글, 한셀, 한쇼 모두 자동 감지.

▲ 전역 토글

⌃⌥⌘P로 어디서든 즉시 on/off.

▲ 안전 설계

암호 입력창(Secure Input)에서는 자동 비활성. 앱이 비활성일 땐 키 변환 안 함.

▲ 자동 업데이트

Sparkle 통합. 버그 수정과 기능 개선이 즉시 전달.

▲ 공증 완료

Apple Developer ID 서명 + 공증. Gatekeeper 경고 없이 바로 설치.

기본 매핑

누르는 키HWP에 전송적용되는 스타일
⌘ 0⌃ 2스타일 목록의 2번째
⌘ 1⌃ 3스타일 목록의 3번째
⌘ 2⌃ 4스타일 목록의 4번째
⌘ 3⌃ 5스타일 목록의 5번째
⌘ 4⌃ 6스타일 목록의 6번째

한컴 오피스는 스타일 이름이 아니라 목록 순서로 단축키를 붙입니다. 원하는 스타일을 ⌘0~4에 배치하려면, HWP 서식 → 스타일에서 목록 순서를 바꾸세요. 앱 설정의 단축키 탭에 5단계 가이드가 있습니다.

설치

  1. 위의 DMG 다운로드 버튼 클릭
  2. DMG 파일 더블클릭 → 한컴단축키.app을 Applications로 드래그
  3. Launchpad에서 한컴단축키 실행
  4. 첫 실행 시 안내에 따라 접근성 권한 부여 (시스템 설정 → 개인정보 보호 및 보안 → 접근성)
  5. 한글, 한셀, 한쇼에서 ⌘ 0~4 눌러 확인

호환

macOS 13 Ventura 이상. Apple Silicon 및 Intel 맥 모두 지원 (Universal binary).

한컴 오피스 버전: v12.30 이상에서 검증. 다른 버전에서도 일반적으로 동작합니다.
